# Local PC bridge quick start

Files:
- `pc_local_bridge.py` -> main local bridge
- `requirements.txt` -> Python packages
- `config.local_only.example.json` -> local-only config example
- `config.hybrid.example.json` -> hybrid config example
- `gamestate_integration_local_bridge.cfg` -> CS2 cfg
- `start_visible_bridge.bat` -> visible Python window
- `start_cs2_led_everything.bat` -> OpenRGB + visible bridge
- `start_cs2_led_everything_hidden.vbs` -> OpenRGB + hidden bridge
- `stop_hidden_bridge.bat`
- `restart_hidden_bridge.bat`
- `install_autostart_shortcut.ps1`
- `uninstall_autostart_shortcut.ps1`
